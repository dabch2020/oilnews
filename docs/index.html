英国海上贸易行动办公室：美国针对伊朗港口限制措施已实施

【英国海上贸易行动办公室：美国针对伊朗港口限制措施已实施】财联社4月14日电，英国海上贸易行动办公室13日发布通告说，自协调世界时13日14时（北京时间13日22时）起，美国针对伊朗港口及沿海区域的海上通行限制措施已经实施。通告说，无论悬挂任何旗帜，只要涉及伊朗港口、石油码头或沿海设施，其海上通行都将受限制措施影响。限制范围涵盖整个伊朗海岸线，包括港口及能源基础设施。通告同时指出，经霍尔木兹海峡、往返非伊朗目的地的过境航行，目前未报告受限制措施的阻碍。但船舶通行期间可能遭遇军事存在、指令性通信或登临检查程序。通告提醒，在霍尔木兹海峡、阿曼湾和邻近海域航行的船舶，应保持高度态势感知。 (央视新闻)

【OPEC 3月产量创史上最大单月跌幅】财联社4月13日电，OPEC数据显示，该组织3月原油产量创下纪录最大单月跌幅，因为中东冲突导致主要成员国出口受限。报告显示，3月该组织产量减少788万桶/日，至2079万桶/日，主要受伊拉克、沙特阿拉伯、阿联酋和科威特产量下降拖累。这是自上世纪80年代有数据以来的最大降幅。报告显示，伊拉克3月产量降幅最大，减少256万桶/日，至163万桶/日。沙特紧随其后，减少231万桶/日，至780万桶/日。
